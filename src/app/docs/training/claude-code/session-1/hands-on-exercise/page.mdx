# Hands-On Exercise

15 minutes — open claude.ai and follow along.

1. Configure your user preferences in Settings > Profile
2. Open Claude Code in your terminal: run `claude` and ask it the same question you asked in chat
3. Compare: same prompt in chat (claude.ai) vs. CLI (claude). Notice the differences
4. Try a real task — draft a runbook, troubleshoot a config, write a KQL query, or review a CA policy — at different thinking effort levels
5. Iterate: take Claude's first output, find something wrong, push back and refine
6. Self-check: append "then review your output for accuracy" to your next request
