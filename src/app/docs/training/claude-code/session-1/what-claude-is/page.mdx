# What Claude Is (and Isn't)
Claude is a reasoning model and versatile thought partner that thinks through problems across code, analysis, writing, and automation, working best when you provide clear context and constraints

It is **NOT** a search engine, a knowledge database, or a replacement for your judgment — vague input produces vague output, and outputs should always be validated.

Treat Claude as a collaborative tool you iterate with, not a magic box that gets everything right on the first try.
