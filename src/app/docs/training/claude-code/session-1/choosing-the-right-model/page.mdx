# Choosing the Right Model

<ReadTime minutes={3} />

Match the model to the task — the biggest lever for managing token consumption.

<br />
<Callout variant="info" title="Rule of thumb:" align="left">
  Start with Sonnet. Escalate to Opus when the task requires deep reasoning or
  you're not getting good results. Drop to Haiku for quick, simple tasks. In
  Claude Code, use **Opus plan mode** — Opus plans the approach, Sonnet executes
  it — best of both at lower cost.
</Callout>
