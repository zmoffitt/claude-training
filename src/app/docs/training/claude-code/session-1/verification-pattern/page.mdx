# The Verification Pattern

Ask Claude to self-check — the single biggest quality improvement you can make.

### Create then review

```text
"...then review it for accuracy and completeness before saving"
```

### Create then test

```text
...then spot-check 5 random rows against the source data"
```

### Create then compare

```text
> _"...then compare it against the original to make sure nothing was missed"_
```

### Multi-step checkpoint

```text
> _"After each section, pause and verify before moving on"_
```
