## How to enable

Settings → Connectors → pick provider → OAuth flow in your browser → scope the permissions you grant. Revoke any time from the same panel or the provider's admin console.

<br />

<Callout variant="warning" title="Mind the scope" align="left">
  Connectors run with **your** identity. Anything Claude reads through them
  follows your existing permissions — no more, no less. Privileged accounts
  should connect from a least-privileged Claude session.
</Callout>
