# Introduction

Welcome to the Cloud AI Training documentation — the central resource for the Platform Engineering team to learn and share knowledge about AI coding tools.

## What you'll find here

- **[Training Materials](/docs/training/claude-code)** — Guides, walkthroughs, and session notes from team trainings
- **[Best Practices](/docs/best-practices/code-review)** — Proven patterns and workflows for using AI coding tools effectively
- **[Configuration](/docs/configuration/claude-md)** — Shared configs, prompts, and tool settings
- **[Tips & Tricks](/docs/tips/effective-prompts)** — Team-contributed knowledge for getting the most out of AI assistants

## Tools covered

This documentation covers AI-assisted development platforms including:

- **Claude** — Anthropic's AI assistant (Claude Code CLI, IDE extensions, API)
- **GitHub Copilot** — AI pair programming in your editor
- **Other AI tools** — Codex, Cursor, and emerging platforms

All Platform Engineering team members are encouraged to contribute. Add your training notes, useful prompts, configuration examples, or any resources that help the team level up with AI tools.
